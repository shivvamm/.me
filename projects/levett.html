01 / Problem

The context.

Levett Consultancy offers tiered IT support (Bronze, Gold, Platinum) to schools. The Platinum tier allows unlimited queries — but support engineers were drowning in repetitive L1 tickets whose answers already existed in their 15-year knowledge base or in official Google/Microsoft support documentation.

The challenge: build a system that understands incoming email queries, auto-responds to L1 tickets, logs time and statuses, and intelligently escalates complex issues — all while meeting the accuracy and compliance standards expected in educational IT.

02 / Solution

Three-layer platform.

01
Email Triage

Workflow extracts query intent, classifies ticket complexity, routes L1 for auto-response and escalates L2/L3 to engineers with full context.

02
Multi-Source RAG

Retrieval across the 15-year internal knowledge base + official Google/Microsoft support docs for contextual, accurate L1 responses.

03
Guardrails

Validation layer ensuring every auto-response is grounded in retrieved context and safe to ship — no hallucinations, no compliance gaps.

03 / My Contributions

What I built.

A.
Email query + escalation pipeline

Workflow orchestration achieving 94% escalation precision (correctly identifying L2/L3) and 92% extraction accuracy for intent classification — cutting manual triage dramatically.

B.
RAG response generation

Built on Vertex AI. Integrated the 15-yr KB with Google/MSFT docs. 87% context precision · 13.64s P99 first-response latency.

C.
Guardrails enforcement

Every auto-generated response context-grounded and safe-to-ship. Minimized manual override interventions for compliance.
