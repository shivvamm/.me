01 / Problem

The context.

1N20, a door and window manufacturer, wanted to transform their website into an intelligent sales channel. Their sales team was drowning in repetitive qualification calls, and customers had no way to get instant, personalized guidance while browsing.

The vision: an always-available AI agent that could engage through voice, capture requirements, present products with visuals, and hand off qualified leads to humans — essentially handling the entire top-of-funnel while maintaining a personal touch.

02 / Solution

Three systems
stitched together.

01
Dual-Agent Orchestration

Data-capture agent hands off to a guided sales agent equipped with 12+ specialized tools, shared memory, and persistent sessions for returning customers.

02
Multimodal Discovery

RAG over 9,000+ product chunks in Pinecone with text-embedding-3-large. Returns product images alongside text for visual presentation mid-conversation.

03
Real-Time Voice

Low-latency WebSocket streaming pipeline. GPT-4o-mini for STT/TTS, Pydantic-validated outputs, automated lead capture into MongoDB with follow-up emails.

03 / My Contributions

What I built.

A.
Architected the dual-agent system

Built with OpenAI Agents SDK. Implemented context-aware handoff from capture → guided-sales agent with 12+ integrated tools, shared memory for conversation continuity, and persistent sessions across multi-visit customers.

B.
Built the multimodal RAG pipeline

Pinecone + text-embedding-3-large, chunked 9,000+ offerings for precise retrieval. Response layer stitches product images into the text response so the agent can "show and tell" in real time.

C.
Developed the voice interface

Low-latency streaming WebSocket pipeline, GPT-4o-mini STT/TTS, Pydantic response schemas, lead-ingestion persisting qualified leads to MongoDB with automated follow-up emails.

04 / Live Trace

A typical session.

$ voice-agent --session=8a2f-c001 --mode=sales
◉ STT stream open · GPT-4o-mini
◉ session::load(customer_id=8a2f) → returning_visitor
▸ customer: "hi, need a 12ft patio door, white, double-glazed"
▸ agent::capture → requirements { width: 12ft, type: patio, color: white, glazing: 2x }
▸ agent.handoff → sales_guide
▸ tools::search_products(q="12ft white patio double-glazed")
▸ RAG::pinecone ← index(products-v4) · top_k=5 · score=0.82
▸ retrieved: patio_door_12ft_white.jpg + 3 spec sheets
▸ agent: "I have 3 options. The Classic Pro at $2,400 is our best seller—"
▸ customer: "sounds good, book someone to measure"
▸ lead.create → mongo.leads::insert({ score: 0.91, hot: true })
▸ email.queue → sales_team@1n20.com
✓ session::persist · duration=4m12s · outcome=QUALIFIED
